# EIP 정보처리 학습센터 - 프로젝트 개요

## 프로젝트 소개
한국산업인력공단의 정보처리 분야 자격증(정보처리기사, 정보처리산업기사, 프로그래밍기능사) 필기/실기 종합 학습 사이트.

## 대상 자격증
| 자격증 | 코드 | 설명 |
|--------|------|------|
| 정보처리기사 | `engineer` | 기사급, 5과목 |
| 정보처리산업기사 | `industrial` | 산업기사급, 4과목 |
| 프로그래밍기능사 | `functional` | 기능사급, 3과목 |

## 주요 기능
1. **자격증 소개** - 3종 자격증 상세 정보 (시험과목, 합격기준, 시험시간)
2. **필기시험 학습** - 과목별 이론 정리, CBT 모의시험
3. **실기시험 학습** - SQL 실습, 알고리즘 연습, 단답형 문제
4. **코딩 실습** - C/Java/Python 온라인 코딩 (Monaco Editor + Wandbox API)
5. **온라인 강의** - 유튜브 강의 연동
6. **마이페이지** - 학습 이력, 시험 결과 통계
7. **커뮤니티** - 질문/답변, 시험 후기, 스터디 모집 (준비 중)

## 네비게이션 구조
- **자격증 소개** - 자격증 개요, 각 자격증별 상세 정보
- **정보처리기사** - 필기시험(5과목+모의시험), 실기시험, 코딩실습, 강의실
- **정보처리산업기사** - 필기시험(4과목+모의시험), 실기시험, 코딩실습, 강의실
- **프로그래밍기능사** - 필기시험(3과목+모의시험), 실기시험, 코딩실습, 강의실
- **커뮤니티** - 준비 중
