# EIP 정보처리 학습센터 - 프로젝트 개요

## 프로젝트 소개
한국산업인력공단의 정보처리 분야 자격증(정보처리기사, 정보처리산업기사, 프로그래밍기능사) 필기/실기 종합 학습 사이트.

## 대상 자격증
| 자격증 | 코드 | 설명 |
|--------|------|------|
| 정보처리기사 | `engineer` | 기사급, 5과목 |
| 정보처리산업기사 | `industrial` | 산업기사급, 4과목 |
| 프로그래밍기능사 | `functional` | 기능사급, 3과목 |

## 주요 기능
1. **필기시험 학습** - 과목별 이론 정리, CBT 모의시험
2. **실기시험 학습** - SQL 실습, 알고리즘 연습, 단답형 문제
3. **코딩 실습** - C/Java/Python 온라인 코딩 (Monaco Editor + Judge0 API)
4. **온라인 강의** - 유튜브 강의 연동
5. **마이페이지** - 학습 이력, 시험 결과 통계
