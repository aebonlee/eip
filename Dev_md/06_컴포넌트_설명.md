# 컴포넌트 설명

## UI 공통 컴포넌트 (`src/components/ui/`)

### Button.jsx
재사용 가능한 버튼 컴포넌트
- **Props**: `variant` (primary/accent/outline/ghost/success/warning/danger), `size` (sm/md/lg/xl), `className`
- **사용 예**: `<Button variant="accent" size="lg">시작하기</Button>`

### Card.jsx
카드 컨테이너 컴포넌트
- **Props**: `hover` (호버 효과), `className`
- **기본 스타일**: `bg-primary-light` 배경, `border-primary-lighter` 테두리

### Modal.jsx
모달 다이얼로그 컴포넌트
- **Props**: `isOpen`, `onClose`, `title`, `size` (sm/md/lg/xl/full)
- 백드롭 클릭으로 닫기, ESC 키 미지원 (추후 추가 가능)

### Timer.jsx
시험 타이머 컴포넌트
- **Props**: `duration` (초), `onTimeUp` (콜백), `isRunning`
- 5분 미만: 노란색, 1분 미만: 빨간색 + 깜빡임

### ProgressBar.jsx
진행 바 컴포넌트
- **Props**: `value`, `max`, `label`, `color` (accent/success/warning/secondary), `size` (sm/md/lg)

## 레이아웃 (`src/components/layout/`)

### Layout.jsx
전체 페이지 래퍼. Header + main(Outlet) + Footer 구성.

### Header.jsx
상단 네비게이션 바.
- 반응형 (모바일 햄버거 메뉴)
- 로그인 상태에 따른 사용자 정보/로그인 버튼 표시

### Sidebar.jsx
사이드 메뉴 (현재 Layout에서 미사용, 필요시 활성화)

### Footer.jsx
하단 저작권 정보

## 인증 (`src/components/auth/`)

### LoginPage.jsx
- Google/Kakao OAuth 로그인 버튼
- 이미 로그인된 경우 홈으로 리다이렉트

### AuthCallback.jsx
- OAuth 인증 후 리다이렉트 처리
- 로그인 성공 시 홈으로 이동

## 필기시험 (`src/components/written-exam/`)

### WrittenExamHome.jsx
자격증 3종 선택 카드 (기사/산업기사/기능사)

### SubjectList.jsx
선택한 자격증의 과목 목록 + CBT 모의시험 바로가기

### ChapterStudy.jsx
과목별 챕터 이론 학습. 좌측 챕터 목록 + 우측 콘텐츠.

### MockTest.jsx
CBT 모의시험 (20문제, 30분 제한). 문제 번호 네비게이션, 타이머, 제출 기능.

### TestResult.jsx
시험 결과 표시. 합격/불합격 판정, 문제별 정답/오답, 해설 보기.

## 실기시험 (`src/components/practical-exam/`)

### SQLPractice.jsx
SQL 구문 작성 연습. 정답 비교, 힌트, 해설 제공.

### AlgorithmPractice.jsx
알고리즘 결과 입력 연습.

### ShortAnswer.jsx
단답형 문제 연습. 진행 상황 표시.

## 코딩실습 (`src/components/coding-lab/`)

### CodeEditor.jsx
Monaco Editor 래퍼 + Judge0 API 코드 실행. base64 인코딩/디코딩 처리.

### CodingExercise.jsx
문제 설명 + 코드 에디터 2컬럼 레이아웃. 힌트, 정답 코드 보기 제공.

## 강의실 (`src/components/lectures/`)

### LecturesHome.jsx
카테고리 필터 + 강의 카드 그리드. YouTube 썸네일 표시.

### LecturePlayer.jsx
YouTube iframe 임베드 + 관련 강의 사이드바.

## 마이페이지 (`src/components/mypage/`)

### MyPage.jsx
3탭 구성: 개요(통계), 시험 결과, 코딩 이력. 로그인 필수.
