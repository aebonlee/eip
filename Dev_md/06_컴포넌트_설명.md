# 컴포넌트 설명

## UI 공통 컴포넌트 (`src/components/ui/`)

### Button.jsx
재사용 가능한 버튼 컴포넌트
- **Props**: `variant` (primary/accent/outline/ghost/success/warning/danger), `size` (sm/md/lg/xl), `className`
- **CSS 클래스**: `.btn`, `.btn-primary`, `.btn-accent`, `.btn-outline` 등

### Card.jsx
카드 컨테이너 컴포넌트
- **Props**: `hover` (호버 효과), `className`
- **CSS 클래스**: `.card`, `.card-hover`

### Modal.jsx
모달 다이얼로그 컴포넌트
- **Props**: `isOpen`, `onClose`, `title`, `size` (sm/md/lg/xl/full)
- 백드롭 클릭으로 닫기

### Timer.jsx
시험 타이머 컴포넌트
- **Props**: `duration` (초), `onTimeUp` (콜백), `isRunning`
- 5분 미만: 노란색, 1분 미만: 빨간색 + 깜빡임

### ProgressBar.jsx
진행 바 컴포넌트
- **Props**: `value`, `max`, `label`, `color` (accent/success/warning/secondary), `size` (sm/md/lg)

## 레이아웃 (`src/components/layout/`)

### Layout.jsx
전체 페이지 래퍼. Header + main(Outlet) + Footer 구성.

### Header.jsx
메가 메뉴 네비게이션 바.
- **5개 상위 메뉴**: 자격증 소개, 정보처리기사, 정보처리산업기사, 프로그래밍기능사, 커뮤니티
- **드롭다운**: 자격증 소개는 심플 드롭다운, 자격증별 메뉴는 메가 드롭다운 (필기/실기/코딩/강의 섹션)
- **데스크톱**: hover로 드롭다운 열기
- **모바일**: 아코디언 방식 접기/펼치기 (햄버거 메뉴)
- **글래스모피즘**: backdrop-filter blur, 스크롤 시 불투명 전환
- **유저 메뉴**: 로그인 시 아바타 + 풍선 드롭다운 (마이페이지/로그아웃)

### Footer.jsx
다크 그라디언트 푸터 (3컬럼).
- **Brand**: 로고 + 설명 + Family Site 셀렉트
- **바로가기**: 자격증 소개, 필기시험, 실기시험, 코딩실습, 강의실, 커뮤니티
- **안내**: 이메일(aebon@dreamitbiz.com), 전화번호(010-3700-0629), 카카오톡(aebon), 운영시간(평일 09:00~18:00)
- **맨 위로 버튼**: fixed position, 스크롤 시 표시

## 코딩실습 (`src/components/coding-lab/`)

### CodeEditor.jsx
Monaco Editor 래퍼 + **Wandbox API** 코드 실행.
- Wandbox 컴파일러: `gcc-head` (C), `openjdk-head` (Java), `cpython-head` (Python)
- Java class명 자동 변환: `public class Main` → `public class prog` (Wandbox 요구사항)
- stdin 입력 지원
- 컴파일 에러 / 런타임 에러 / stdout 분리 표시

### CodingExercise.jsx
문제 설명 + 코드 에디터 2컬럼 레이아웃. 힌트, 정답 코드 보기 제공.
