# Supabase DB 스키마

모든 테이블은 `sb_` 접두사를 사용합니다.

## 테이블 목록

### sb_profiles (사용자 프로필)
```sql
CREATE TABLE sb_profiles (
  id UUID REFERENCES auth.users PRIMARY KEY,
  nickname TEXT,
  avatar_url TEXT,
  target_cert TEXT,  -- 목표 자격증 (engineer/industrial/functional)
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```

### sb_study_progress (학습 진도)
```sql
CREATE TABLE sb_study_progress (
  id BIGSERIAL PRIMARY KEY,
  user_id UUID REFERENCES auth.users,
  cert_type TEXT,      -- engineer/industrial/functional
  subject_id TEXT,     -- 과목 ID
  chapter_id TEXT,     -- 챕터 ID
  completed BOOLEAN DEFAULT FALSE,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

### sb_test_results (시험 결과)
```sql
CREATE TABLE sb_test_results (
  id BIGSERIAL PRIMARY KEY,
  user_id UUID REFERENCES auth.users,
  cert_type TEXT,      -- 자격증 유형
  test_type TEXT,      -- written/practical
  score INTEGER,       -- 맞은 문제 수
  total INTEGER,       -- 전체 문제 수
  answers JSONB,       -- {questionId: selectedOption} 형태
  taken_at TIMESTAMPTZ DEFAULT NOW()
);
```

### sb_code_submissions (코딩 제출 기록)
```sql
CREATE TABLE sb_code_submissions (
  id BIGSERIAL PRIMARY KEY,
  user_id UUID REFERENCES auth.users,
  language TEXT,       -- c/java/python
  exercise_id TEXT,    -- 문제 ID
  code TEXT,           -- 제출한 코드
  result TEXT,         -- 실행 결과
  is_correct BOOLEAN,  -- 정답 여부
  submitted_at TIMESTAMPTZ DEFAULT NOW()
);
```

## Supabase 헬퍼 함수 (`src/lib/supabase.js`)

| 함수명 | 용도 |
|--------|------|
| `sb_getProfile(userId)` | 프로필 조회 |
| `sb_upsertProfile(profile)` | 프로필 생성/수정 |
| `sb_getStudyProgress(userId, certType)` | 학습 진도 조회 |
| `sb_updateStudyProgress(progress)` | 학습 진도 업데이트 |
| `sb_getTestResults(userId)` | 시험 결과 목록 조회 |
| `sb_saveTestResult(result)` | 시험 결과 저장 |
| `sb_getCodeSubmissions(userId)` | 코딩 제출 이력 조회 |
| `sb_saveCodeSubmission(submission)` | 코딩 제출 저장 |

## RLS (Row Level Security) 권장 설정
```sql
-- sb_profiles: 본인만 조회/수정
ALTER TABLE sb_profiles ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Users can view own profile" ON sb_profiles FOR SELECT USING (auth.uid() = id);
CREATE POLICY "Users can update own profile" ON sb_profiles FOR UPDATE USING (auth.uid() = id);
CREATE POLICY "Users can insert own profile" ON sb_profiles FOR INSERT WITH CHECK (auth.uid() = id);

-- sb_test_results: 본인만 조회/삽입
ALTER TABLE sb_test_results ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Users can view own results" ON sb_test_results FOR SELECT USING (auth.uid() = user_id);
CREATE POLICY "Users can insert own results" ON sb_test_results FOR INSERT WITH CHECK (auth.uid() = user_id);
```
