# 기술 스택

## Frontend
| 기술 | 버전 | 용도 |
|------|------|------|
| Vite | 8.x | 빌드 도구 |
| React | 19.x | UI 프레임워크 |
| React Router | 7.x | SPA 라우팅 |
| 순수 CSS | - | CSS Custom Properties 기반 디자인 시스템 |
| Monaco Editor | 4.7.x | 코드 에디터 |

## CSS 모듈 구조
| 파일 | 용도 |
|------|------|
| `styles/base.css` | CSS 변수, 리셋, 공통 유틸리티 |
| `styles/navbar.css` | 글래스모피즘 네비바 + 메가 메뉴 드롭다운 |
| `styles/hero.css` | 히어로 섹션, 파티클 애니메이션 |
| `styles/footer.css` | 다크 그라디언트 푸터 |
| `styles/course.css` | 카드, 퀴즈, 페이지헤더 등 학습 UI |
| `styles/editor.css` | 코드 에디터, 실행결과 |
| `styles/lectures.css` | 강의실 레이아웃 |
| `styles/mypage.css` | 마이페이지 탭, 통계 |
| `styles/animations.css` | 키프레임 애니메이션 |

## Backend / Services
| 서비스 | 용도 |
|--------|------|
| Supabase | PostgreSQL DB, Auth, Storage |
| Wandbox API | 온라인 코드 실행 (C/Java/Python) - 무료, 키 불필요, CORS 지원 |
| YouTube Embed | 강의 영상 재생 |

## 인증
- Google OAuth (Supabase Auth)
- Kakao OAuth (Supabase Auth)
