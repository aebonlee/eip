# 기술 스택

## Frontend
| 기술 | 버전 | 용도 |
|------|------|------|
| Vite | 8.x | 빌드 도구 |
| React | 19.x | UI 프레임워크 |
| React Router | 7.x | SPA 라우팅 |
| TailwindCSS | 4.x | CSS 프레임워크 |
| Monaco Editor | 4.7.x | 코드 에디터 |
