# 환경설정 가이드

## 1. 프로젝트 설치

```bash
cd D:\dreamit-web\eip
npm install
```

## 2. 환경변수 설정 (.env)

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_JUDGE0_API_URL=https://judge0-ce.p.rapidapi.com
VITE_JUDGE0_API_KEY=your_rapidapi_key
```

### Supabase 설정
1. [Supabase](https://supabase.com) 프로젝트 생성
2. Project Settings > API에서 URL과 anon key 복사
3. Authentication > Providers에서 Google, Kakao 활성화
4. Redirect URL: `https://eip.dreamitbiz.com/auth/callback`

### Judge0 API 설정
1. [RapidAPI](https://rapidapi.com)에서 Judge0 CE 구독 (무료 플랜 가능)
2. API Key를 .env에 설정

## 3. Supabase DB 테이블 생성

Supabase SQL Editor에서 실행:

```sql
CREATE TABLE sb_profiles (
  id UUID REFERENCES auth.users PRIMARY KEY,
  nickname TEXT,
  avatar_url TEXT,
  target_cert TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE sb_study_progress (
  id BIGSERIAL PRIMARY KEY,
  user_id UUID REFERENCES auth.users,
  cert_type TEXT,
  subject_id TEXT,
  chapter_id TEXT,
  completed BOOLEAN DEFAULT FALSE,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE sb_test_results (
  id BIGSERIAL PRIMARY KEY,
  user_id UUID REFERENCES auth.users,
  cert_type TEXT,
  test_type TEXT,
  score INTEGER,
  total INTEGER,
  answers JSONB,
  taken_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE sb_code_submissions (
  id BIGSERIAL PRIMARY KEY,
  user_id UUID REFERENCES auth.users,
  language TEXT,
  exercise_id TEXT,
  code TEXT,
  result TEXT,
  is_correct BOOLEAN,
  submitted_at TIMESTAMPTZ DEFAULT NOW()
);
```

## 4. 개발 서버 실행

```bash
npm run dev
```
→ http://localhost:5173

## 5. 프로덕션 빌드

```bash
npm run build
```
→ `dist/` 폴더에 빌드 결과물 생성

## 6. OG 이미지 생성

```bash
npm run generate-og
```
→ `public/og-image.png` 생성 (1200x630)

## 7. 배포

### Vercel 배포
