# 라우팅 구조

## 전체 라우트 맵

| 경로 | 컴포넌트 | 설명 |
|------|----------|------|
| `/` | `HomePage` | 랜딩/대시보드 |
| `/login` | `LoginPage` | 소셜 로그인 (Google/Kakao) |
| `/auth/callback` | `AuthCallback` | OAuth 리다이렉트 처리 |
| `/cert-intro` | `CertIntroPage` | 자격증 소개 (3종 상세 정보) |
| `/community` | `CommunityPage` | 커뮤니티 (준비 중) |
| `/written-exam` | `WrittenExamHome` | 필기시험 - 자격증 선택 |
| `/written-exam/:certType` | `SubjectList` | 과목 목록 |
| `/written-exam/:certType/:subjectId/study` | `ChapterStudy` | 챕터별 이론 학습 |
| `/written-exam/:certType/mock-test` | `MockTest` | CBT 모의시험 |
| `/written-exam/:certType/result` | `TestResult` | 시험 결과 |
| `/practical-exam` | `PracticalExamHome` | 실기시험 홈 (기본: 기사) |
| `/practical-exam/:certType` | `PracticalExamHome` | 자격증별 실기시험 홈 |
| `/practical-exam/:certType/sql` | `SQLPractice` | 자격증별 SQL 실습 |
| `/practical-exam/:certType/algorithm` | `AlgorithmPractice` | 자격증별 알고리즘 실습 |
| `/practical-exam/:certType/short-answer` | `ShortAnswer` | 자격증별 단답형 연습 |
| `/coding-lab` | `CodingLabHome` | 코딩실습 홈 |
| `/coding-lab/:language` | `CodingExercise` | 언어별 코딩실습 (c/java/python) |
| `/lectures` | `LecturesHome` | 강의 목록 |
| `/lectures/:id` | `LecturePlayer` | 강의 재생 |
| `/mypage` | `MyPage` | 학습 이력/통계 |
| `*` | 404 | 페이지 없음 |

## URL 파라미터

### certType (자격증 유형)
- `engineer` - 정보처리기사
- `industrial` - 정보처리산업기사
- `functional` - 프로그래밍기능사

### subjectId (과목 ID)
- 기사: `sw-design`, `sw-dev`, `db`, `sys-integration`, `info-security`
- 산업기사: `info-sys`, `programming`, `db`, `info-comm`
- 기능사: `programming-basic`, `algorithm`, `db-basic`

### language (프로그래밍 언어)
- `c` - C언어 (Wandbox: gcc-head)
- `java` - Java (Wandbox: openjdk-head)
- `python` - Python (Wandbox: cpython-head)

## 라우팅 구조 (React Router v7)
- 최상위: `<BrowserRouter>` in `main.jsx`
- 모든 페이지: `<Layout>` 래퍼 (Header + Main + Footer)
- `<Outlet>` 패턴으로 중첩 라우팅

## 네비게이션 메뉴 구조 (메가 메뉴)
```
자격증 소개 (드롭다운)
├── 자격증 개요

└── 프로그래밍기능사

정보처리기사 (메가 드롭다운)
├── 필기시험
│   ├── 과목 목록

│   ├── 프로그래밍 언어 활용
│   ├── 정보시스템 구축관리
│   └── 모의시험
├── 실기시험
│   ├── SQL 실습
│   ├── 알고리즘
│   └── 단답형 연습
├── 코딩실습
│   ├── C언어
│   ├── Java
│   └── Python
└── 강의실
    └── 전체 강의

정보처리산업기사 (메가 드롭다운)

│   ├── 프로그래밍 언어 활용

프로그래밍기능사 (메가 드롭다운)

│   ├── 과목 목록

│   ├── 알고리즘
│   ├── 데이터베이스
│   └── 모의시험
├── 실기시험 (/practical-exam/functional/*)
├── 코딩실습 / 강의실

커뮤니티 (단일 링크)
```
