## 2. Seeded Shuffle 알고리즘

### 목적
같은 회차 번호 → 항상 같은 문제 조합을 보장합니다.

### 구현 (`src/data/round-utils.js`)

```javascript
// djb2 해시 함수
function hashString(str) {
  let hash = 0
  for (let i = 0; i < str.length; i++) {
    const char = str.charCodeAt(i)
    hash = ((hash << 5) - hash) + char
    hash = hash & hash  // 32bit 정수 변환
  }
  return Math.abs(hash)
}

// LCG(Linear Congruential Generator) 기반 셔플
function seededShuffle(array, seed) {
  const arr = [...array]
  let s = seed
  const next = () => {
    s = (s * 1664525 + 1013904223) & 0x7fffffff
    return s / 0x7fffffff
  }
  // Fisher-Yates 셔플
  for (let i = arr.length - 1; i > 0; i--) {
    const j = Math.floor(next() * (i + 1))
    ;[arr[i], arr[j]] = [arr[j], arr[i]]
  }
  return arr
}
```

### 시드 생성 규칙

| 시험 유형 | 시드 문자열 |
|-----------|------------|
| 필기 | `written-${certType}-round-${roundNumber}` |
| 실기 SQL | `practical-${certType}-round-${roundNumber}` (seed) |
| 실기 알고리즘 | seed + 1 |
| 실기 단답형 | seed + 2 |

## 3. 회차별 문제 출제

### 필기시험

- **문제 수**: 회차당 20문제
- **최대 회차**: `Math.floor(전체문제수 / 20)`
  - 기사: 200 / 20 = **10회차**
  - 산업기사: 160 / 20 = **8회차**
  - 기능사: 120 / 20 = **6회차**
- **함수**: `getWrittenRoundQuestions(certType, roundNumber, count=20)`

### 실기시험

- **문제 구성**: SQL 7 + 알고리즘 6 + 단답형 7 = **20문제**
- **최대 회차**: `Math.floor(전체문제수 / 20)`
- **함수**: `getPracticalRoundQuestions(certType, roundNumber)`
- **출력 형식**: 각 문제에 `type` 필드 추가 (`'sql'`, `'algorithm'`, `'short'`)

## 5. 합격 예측 알고리즘

### EWMA (Exponentially Weighted Moving Average)

최신 시험 결과에 높은 가중치를 부여하여 합격 확률을 계산합니다.

```
확률 = 최신점수 × 0.5 + 직전점수 × 0.3 + 나머지평균 × 0.2
```

### 추세 분석 (Linear Trend)

최근 5회 시험 결과에 선형회귀를 적용합니다.

| 기울기 | 판정 | 라벨 |
|--------|------|------|
| > 2 | 상승 | 상승 추세 |
| < -2 | 하락 | 하락 추세 |
| 그 외 | 유지 | 유지 중 |
| 데이터 < 3개 | - | 데이터 부족 |

### 약점 분석

가장 최근 시험의 `subject_scores`에서 60% 미만인 과목/유형을 추출하여 표시합니다.

## 7. Supabase 데이터 모델

### sb_test_results 테이블

| 필드 | 타입 | 설명 |
|------|------|------|
| id | uuid | PK |
| user_id | uuid | FK → auth.users |
| cert_type | text | `'engineer'` / `'industrial'` / `'functional'` |
| test_type | text | `'written'` / `'practical'` |
| round_number | integer | 회차 번호 (null = 랜덤) |
| score | integer | 맞은 문제 수 |
| total | integer | 총 문제 수 |
| subject_scores | jsonb | 과목/유형별 점수 `{ subjectId: { correct, total } }` |
| taken_at | timestamptz | 응시 시각 |

### API 함수

```javascript
// 결과 저장
sb_saveTestResult({ user_id, cert_type, test_type, round_number, score, total, subject_scores })

// 유형별 결과 조회 (합격 예측용)
sb_getTestResultsByType(userId, certType, testType)
// → cert_type + test_type 필터, taken_at DESC 정렬
```

## 10. 컴포넌트 상세

### RoundSelect / PracticalRoundSelect

- 회차 카드 그리드 표시 (1회~N회)
- Supabase에서 이전 시험 결과 로드하여 합격/불합격/미응시 상태 표시
- "랜덤 모의시험" 배너 포함

### PracticalMockTest

- 60분 타이머 (시간 초과 시 자동 제출)
- 문제 유형별 입력 UI:
  - SQL → `<textarea>` (5줄)
  - 알고리즘/단답형 → `<input type="text">`
- 테이블 스키마 표시 (SQL 문제)
- 문제 번호 네비게이션 그리드

### PassPrediction

- 필기/실기 탭 전환
- SVG 반원 게이지 (합격 확률)
- 점수 이력 바 차트 (최근 10회)
- 60% 합격선 점선 표시
- 약점 과목 리스트 (60% 미만)
